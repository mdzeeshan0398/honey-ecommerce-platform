# Honey E-Commerce Platform

A full-stack **e-commerce application for honey products** built using **React, Spring Boot, and MySQL**.  
The platform includes secure authentication, online payments, and complete product & order management.

---

## Features

- User authentication & authorization using **Spring Security + JWT**
- Secure **Razorpay payment gateway** integration
- Product management (Add, Update, Delete, View)
- Order management & order status tracking
- RESTful APIs with Spring Boot
- Responsive frontend built with React
- MySQL database integration

---

## Tech Stack

### Frontend
- React
- Axios
- HTML, CSS, JavaScript

### Backend
- Spring Boot
- spring data jpa
- Spring Security
- JWT Authentication
- REST APIs

### Database
- MySQL

### Payment
- Razorpay
